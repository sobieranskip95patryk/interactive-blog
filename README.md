# Interactive Blog

Prosty, interaktywny blog w HTML, CSS i JavaScript.

## Funkcje
- Dodawanie postów bez przeładowania strony
- Prosty, nowoczesny wygląd

## Struktura
- `index.html` — główny plik strony
- `style.css` — stylizacja
- `script.js` — logika interakcji
- `assets/` — obrazy (cover.jpg, avatar.jpg — opcjonalnie)

## Jak używać
1. Otwórz `index.html` w przeglądarce.
2. Dodaj post przez formularz.

---

Możesz dodać własne obrazy do folderu `assets`.
